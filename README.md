# euler

A collection of [http://www.projecteuler.net](Project Euler) solutions.

# Setup

This project requires ruby \>= 1.9

```
$ git clone https://pgroudas@github.com/pgroudas/euler.git
$ cd ./euler
$ bundle install
$ ./bin/euler
```
